# Experiment List - Short summaries

## Directory Digester

Create a CLI tool that recursively scans a directory and calculates hashes (e.g., SHA1, SHA256) for all files.

### Success Criteria

- CLI accepts a target directory argument.
- CLI accepts a hash algorithm argument (default: sha256).
- Output displays file paths and their corresponding hashes.
- Handles permission errors gracefully (skips unreadable files).
