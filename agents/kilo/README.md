# Kilo Code Setup

Setup and configuration for [Kilo Code](https://kilocode.ai/) - an AI coding
assistant.

- Introduction: <https://app.kilo.ai/welcome>
  - YouTube: <https://www.youtube.com/watch?v=WqDm6Yg1bu0>

## Installation

### VSCode/Antigravity Extension

Kilo Code is available as a VSCode extension. Install it from the VSCode
marketplace or via Antigravity.

### CLI Installation

```bash
npm install -g @kilocode/cli

$ kilo --version
0.26.1

$ kilocode --version
0.26.1
```
