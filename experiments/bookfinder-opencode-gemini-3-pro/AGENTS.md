# Agent Guidelines for bookfinder-opencode-gemini-3-pro

## Workflow

- Run `bun run ci` after completing meaningful tasks to ensure quality.
- Fix any lint/type/test errors immediately.

## Dependencies

- Use `bun add` or `bun add -d`. Do not edit `package.json` manually for deps.

## Constraints

- **Runtime**: Bun only.
- **CLI**: Commander only.
- **TUI**: OpenTUI.
