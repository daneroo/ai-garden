## Live Timer

The Timer below is a React component imported from the workspace:

<Timer client:load initialSeconds={30} />

## Features

- **Countdown**: Starts at the specified seconds
- **Pause/Resume**: Click to toggle
- **Color States**: Red (running), Green (complete)
- **Tailwind Styled**: Uses workspace Tailwind classes
